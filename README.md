# todo-app-handson

タスク管理アプリケーションのハンズオンプロジェクトです。

## 概要

シンプルなタスク管理アプリケーション。ブラウザだけで動作し、サーバーやデータベースは不要です。

## 技術スタック

| 技術 | 用途 |
|------|------|
| HTML | 画面の構造・マークアップ |
| CSS | スタイリング・レイアウト |
| JavaScript | アプリのロジック・DOM操作 |
| LocalStorage | タスクデータのブラウザ内永続化 |

## 機能

- タスクの追加
- タスクの完了/未完了の切り替え
- タスクの削除

## 使い方

1. リポジトリをクローン
   ```bash
   git clone https://github.com/norimine11/todo-app-handson.git
   cd todo-app-handson
   ```
2. `index.html` をブラウザで開く

サーバー不要でそのまま動作します。
